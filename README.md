# Tipdora.fun - Multi Calculator Website

A fast, SEO-optimized, Multi-Calculator website built with Vanilla JavaScript and Tailwind CSS.

## Features
- **12+ Calculator Tools** (BMI, EMI, Age, Crypto, etc.)
- **Fast Performance:** Scores 95+ on Lighthouse.
- **Mobile First:** Fully responsive design.
- **AdSense Ready:** Dedicated ad spots and legal pages.
- **Privacy Focused:** Client-side calculations.

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start Dev Server:
   ```bash
   npm run dev
   ```
3. Build for Production:
   ```bash
   npm run build
   ```

## Folder Structure
- `index.html`: Homepage
- `src/`: JS modules and CSS
- `calculators/`: Individual tool pages
- `blog/`: Blog template
