How GPA is Calculated

Formula: Total Grade Points / Total Credits

Each grade is assigned a number value (e.g., A = 4.0, B = 3.0). This point value is multiplied by the credits for that course.

Example

  • Math (3 Credits): Got A (4.0) -> 3 * 4.0 = 12 Points
  • History (2 Credits): Got B (3.0) -> 2 * 3.0 = 6 Points
  • Total: 18 Points / 5 Credits = 3.6 GPA
